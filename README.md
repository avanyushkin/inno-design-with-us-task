## Interior Design Website

A responsive interior design website built with HTML and CSS, featuring modern design elements and fully responsive layouts.

## Features Implemented

- Responsive Design: Fully responsive across mobile, tablet, and desktop breakpoints
- Pixel-Perfect Implementation: Precise alignment with design specifications at 1440px desktop view
- Component-Based Structure: Organized CSS with modular components

## Local Setup Instructions

### Prerequisites

- ![Node.js](https://img.shields.io/badge/Node.js-14+-339933?style=flat-square&logo=nodedotjs&logoColor=white) (v14 or higher)
- ![npm](https://img.shields.io/badge/npm-6+-CB3837?style=flat-square&logo=npm&logoColor=white) (usually comes with Node.js)

### Installation Steps

1. Clone or download the project files
2. Install dependencies
```bash
npm install express
```
3. Build the CSS bundle
```bash
node build.js
```
This will generate public/bundle.css by combining all component CSS files.
4. Start the development server
```bash
node server.js
```
5. Open your browser and navigate to
```bash
http://localhost:3000
```
